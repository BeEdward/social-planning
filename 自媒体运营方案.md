## 一、人设定位 & 差异化

### 核心人设标签
- **"烧光9位数的90后"** — 这是你最大的钩子，极度稀缺，自带话题性
- **AI实战派** — 不是教授/研究员，是真金白银砸出来的从业者
- **连续创业者** — 见过生死，有真实的商业判断力

### 差异化优势
市面上AI博主大多是：技术科普型、工具测评型、教程搬运型。你的差异点在于：

1. **有真实的商业失败经历** — 这是绝大多数博主不具备的内容护城河
2. **有AI行业一线视角** — 不是纸上谈兵，是真正做过产品的人
3. **敢说真话** — 烧过钱的人才知道什么是坑，这种"过来人"视角极度吸引人

```
用真金白银换来的经验，教你用AI搞钱 & 避坑
```

## 三、平台策略

### 小红书策略

**内容形式优先级：**
1. **图文笔记** — 主力，发布频率高，小红书核心形式
2. **短视频（60s内）** — 创业金句、AI技巧速览
3. **直播** — 中后期再做，用于深度互动和变现

**小红书爆款公式：**
```
标题 = 数字 + 痛点/好奇心 + 身份标签
例：
- "烧光1个亿后，我总结了创业者最容易犯的5个错"
- "90后创业者：用AI一个月省了20万人力成本"
- "融资9位数全亏光，现在我靠AI月入XX万"
```

**封面设计原则：**
- 大字报风格，3秒能看懂
- 用数字、对比、悬念
- 人物出镜增加信任感

**发布节奏：** 每天1条（日更），利用AI批量备稿降低日常压力

**关键策略：**
- 评论区必回，前30分钟是黄金互动期
- 多用小红书的话题标签：#AI搞钱 #创业 #90后创业 #副业 #AI工具
- 积极参与同领域博主的评论区（刷脸）
- 建立粉丝群，提升粘性

### B站策略

**内容形式优先级：**
1. **中视频（5-15分钟）** — 深度内容，B站核心，每周1条
2. **短视频/动态** — 从小红书内容或中视频切片而来，低成本补充
3. **专栏文章** — 技术教程的文字版，从小红书图文改写

**B站爆款元素：**
- 真诚、有料、有深度
- 开头5秒必须有钩子（"我烧了1个亿买来的教训"）
- 适度自嘲和幽默
- 信息密度要高

**发布节奏：** 每周1条中视频（周末集中制作），平时发动态/短视频复用小红书内容

**关键策略：**
- 充分利用B站的长尾流量（搜索SEO）
- 标题和标签要覆盖搜索关键词
- 做系列内容，提升完播率和追更率
- 联动其他UP主（互推）
- 一条中视频可以切3-5条短视频/小红书帖子，反向复用

### 跨平台复用矩阵（省时核心）

一份素材，多平台分发，最大化时间投入产出比：

```
B站中视频（周末制作1条）
  ├─ 切片 → 3-5条小红书短视频/图文
  ├─ 提取金句 → 小红书文字帖
  ├─ 提取脚本 → B站专栏/公众号文章
  └─ 评论区精彩问答 → 下一条内容的选题

小红书日更图文
  ├─ 爆款帖子 → 扩展为B站中视频选题
  ├─ 系列帖子 → 合集整理为长视频
  └─ AI批量生成备稿（周末集中准备5-7条）
```

### 4.3 爆款选题发现技巧

**方法1：跨平台搬运热点**
- 监控Twitter/X上的英文AI社区热点
- 第一时间翻译+加上你的解读发到小红书/B站
- 信息差就是流量差

**方法2：竞品内容分析**
- 监控同领域博主的爆款内容（点赞>1000的笔记）
- 分析爆款共性：标题、封面、话题、发布时间

**方法3：评论区挖金矿**
- 监控热门帖子的评论区
- 高赞评论往往就是下一个爆款选题
- 用户的提问 = 你的内容选题

**方法4：关键词趋势追踪**
- 监控关键词的搜索量变化
- 上升趋势的关键词 = 即将爆发的话题
- 提前布局内容，吃到搜索流量红利

**方法5：AI自动筛选**
- 用AI对监控推送内容做自动分类和打分
- 设置规则：涉及XX关键词 + 互动量超XX = 高优先级推送
- 减少人工筛选时间

## 五、冷启动 & 快速涨粉策略

### 第一阶段：冷启动（第1-2周）

**目标：** 发出前20条内容，找到数据反馈最好的方向

1. **话题炸弹开路** — 第一条内容就要炸
   - 推荐首发："我是怎么烧光1个亿的"
   - 这种内容自带传播性，不需要粉丝基础也能上热门

2. **日更试错** — 每天发1条，不同类型轮换
   - 创业故事 → AI工具 → 搞钱技巧 → 热点 循环
   - 观察数据，找到你的"爆款基因"

3. **平台规则利用**
   - 新号有流量扶持期，必须抓住
   - 小红书新号前10条笔记有额外推荐
   - B站新人奖励计划

### 第二阶段：放量（第3-8周）

**目标：** 做到小红书1万粉，B站5000粉

1. **系列化内容** — 把验证过的爆款方向做成系列
2. **互动拉满** — 每条内容的评论区都积极回复
3. **矩阵账号** — 考虑做一个"创业版"和一个"AI教程版"分开运营
4. **跨平台导流** — 小红书内容同步到B站动态，反之亦然
5. **蹭热点不断** — 每周至少跟2个热点

### 第三阶段：加速（第9-16周）

**目标：** 小红书5万粉，B站2万粉

1. **合作互推** — 找同量级博主互相推荐
2. **投流测试** — 对爆款内容追投DOU+（小红书）/ 推广（B站）
3. **建立私域** — 引导粉丝加群/关注公众号
4. **开始变现** — 接AI工具推广、做付费课程预热

## 七、AI辅助内容生产（时间有限的核心策略）

你的时间有限，但你是AI从业者——**用AI做内容本身就是最好的内容**。
以下是一套用AI最大化产出的体系：

### 7.1 AI可以直接产出的内容类型

这些内容AI可以"几乎全自动"生成，你只需审核+加上个人观点：

| 内容类型 | AI产出方式 | 你需要做的 | 预计耗时 |
|----------|-----------|-----------|----------|
| AI工具测评图文 | 给Claude/GPT一个工具名，生成结构化测评 | 加上你的真实使用感受，截几张图 | 15分钟 |
| 热点快评 | 监控系统推送热点 → AI生成点评初稿 | 加入你的创业者视角，改几句话 | 10分钟 |
| "AI搞钱"教程 | AI生成步骤教程 + 你录屏操作过程 | 录屏+简单校对 | 20分钟 |
| 金句/观点帖 | 把你的一句话想法给AI扩展成图文 | 提供核心观点（一句话即可） | 5分钟 |
| 行业数据整理 | AI汇总+可视化 | 审核数据准确性 | 10分钟 |

### 7.2 AI生成"好玩内容"的玩法（自带传播性）

这些内容本身就展示了AI的能力，与你的人设完美契合：

**玩法1："我让AI帮我复盘创业失败"**
- 把你的创业经历喂给Claude，让它分析你哪步走错了
- 录屏整个过程，AI的"毒舌点评"就是内容
- 小红书图文 + B站视频双发
- **为什么好玩：** 人们既想看AI的分析，又想看你的反应

**玩法2："AI模拟投资人面试我"**
- 让AI扮演投资人，你来pitch
- AI会问出很尖锐的问题
- 真实互动过程就是内容
- **为什么好玩：** 互动感强，评论区会讨论"AI说得对不对"

**玩法3："用AI 1小时做一个产品"系列**
- 从idea到上线，全程录屏
- 展示AI编程、AI设计、AI文案的全流程
- 每期做一个不同的小产品
- **为什么好玩：** 实操性极强，观众可以跟着做

**玩法4："AI帮我算这个生意能不能做"**
- 粉丝提交创业想法，你用AI做商业分析
- AI生成市场分析、竞品分析、财务模型
- 你补充实战经验点评
- **为什么好玩：** 互动性强，粉丝有参与感，选题永远不缺

**玩法5："今日AI圈又出大事了"AI自动生成版**
- 监控系统收集当日AI热点
- AI自动整理成"AI日报"风格的图文
- 你加一句点评就能发
- **为什么好玩：** 几乎零成本日更，但信息价值高

**玩法6："让AI给我的老项目写墓志铭"**
- 把你每个失败项目的情况告诉AI
- AI写出诙谐/深刻的"项目墓志铭"
- 既搞笑又有反思价值
- **为什么好玩：** 创意独特，情绪价值高，自带转发属性

**玩法7："AI vs 我：谁的商业判断更准"**
- 拿出历史上你做过的决策
- 让AI在同样信息下做决策
- 对比结果
- **为什么好玩：** 悬念感强，人机对比自带话题

### 7.3 批量备稿工作流（周末2小时 = 一周小红书）

```

   └─ 逐条过，加入真实细节和个人感受
   └─ 选标题、调语气、删AI味太重的句子
   └─ 关键：加入只有你知道的真实经历/数据/感受

4. 封面批量制作（20分钟）
   └─ 用Canva批量套模板
   └─ 或用AI生图工具批量生成

5. 定时发布设置（10分钟）
   └─ 设置每天定时发布
   └─ 预留1-2个"空位"给临时热点
```

### 7.4 B站视频省时生产方案

每周只需投入 **3-4小时** 产出1条中视频：

```
方案A：口播+屏幕录制（最省时）
  └─ AI写脚本（30分钟审核修改）
  └─ 对着脚本口播录制（30分钟，不用背，可以看提词器）
  └─ 剪辑（1.5小时，用剪映自动字幕）
  └─ 总计：约2.5小时

方案B：AI生成画面+配音（不出镜）
  └─ AI写脚本 → AI生成配图/动画 → TTS或自己配音
  └─ 适合技术教程类内容
  └─ 总计：约2小时（但缺少人设建立）

方案C：录屏实操（最轻松）
  └─ 直接录你使用AI工具的过程
  └─ 边操作边讲解，后期轻剪
  └─ "用AI 1小时做XX"这类内容天然适合
  └─ 总计：实操时间 + 30分钟剪辑

推荐：前期用方案A/C建立人设，穿插方案B保持更新频率
```

### 7.5 小红书图文SOP（目标：15-30分钟/条）

```
1. 选题确认（3分钟）
   └─ 从备稿库选择 or 从监控推送中选择临时热点

2. 内容生成（10分钟）
   └─ 如果是备稿：直接取出AI初稿，快速润色
   └─ 如果是临时：丢给AI生成，加上你的观点
   └─ 核心：加入1-2个只有你知道的真实细节

3. 封面制作（5分钟）
   └─ 套Canva模板 or AI生图
   └─ 大字+对比色

4. 发布 & 互动（5分钟）

   └─ 前30分钟回复评论（可以用AI辅助回复草稿）
```
